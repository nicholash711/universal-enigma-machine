# Universal-Enigma-Machine
## Simulation of multiple German Enigma Machine
---
## Implemented Models
- Abwehr Enigma G-111
- Abwehr Enigma G-260
- Abwehr Enigma G-312
- Enigma D / K
- Enigma I
- Enigma I "Norenigma"
- Enigma I "Sondermaschine"
- Enigma M3
- Enigma M4
- Enigma Swiss-K
- Enigma T "Tirpitz"
- Railway Enigma K
- Zahlwerk Enigma A28

### Does not Support:
- UKW-D
- Uhr

Notice: wxWidgets is required and currently not included in the repository
